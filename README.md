`kelunik/string` is a small library containing string helper functions.

 - `Kelunik\String\starts_with`
 - `Kelunik\String\ends_with`
 - `Kelunik\String\contains`

## Requirements

- PHP 7.1

## Installation

```bash
composer require kelunik/string
```
